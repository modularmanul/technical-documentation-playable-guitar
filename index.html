Introduction

This is a beginner level project to have some fun with what I've learned so far. I will be creating a guitar entirely with HTML, CSS & little-to-no JavaScript.

This is what we will be building. And nope, it's not poop or chocolate ice cream. You can play around (try strumming) first to get into your vibe!

What You Should Already Know

Not much really. As long as you have a working knowledge of how HTML, CSS and JavaScript works together.

Learning Outcomes
  • Learn how to create a playable instrument using HTML, CSS and JavaScript.
  • Learn how to use the audio element in HTML and play sounds using JavaScript.
  • Learn how to use various CSS properties to create a three dimensional look.
  • Have your very own custom-made Guitar with just some tweaks to align with your likings!

Lets Hear It

How about the sound?

Alright. We finally got to this part that you've been looking forward to. Lets add the audio function script directly inside the head element of HTML. Normally it is more efficient to create a separate JavaScript file and link it using the script tag but all the script we will be using is the one below so lets go ahead with this method.

Notice that I included an error handler too just in case it doesn't work as expected, I could check the log for details. 


function play(audioSrc) {
  let audio = new Audio(audioSrc);
  audio.play().catch((e) => console.error("Error playing audio:", e));
}
